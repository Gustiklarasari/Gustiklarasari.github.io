Materi PHP

PHP adalah bahasa pemrograman sisi server yang kuat dan serbaguna, sering digunakan untuk mengembangkan situs web dinamis.

Pelajari tentang dasar-dasar PHP, kontrol alur, fungsi, pemrograman berorientasi objek, dan banyak lagi.

Materi MySQL

MySQL adalah sistem manajemen basis data yang populer dan sering digunakan bersama dengan PHP untuk mengelola data pada situs web.

Explore konsep dasar basis data, kueri SQL, operasi CRUD (Create, Read, Update, Delete), dan lainnya.

Materi CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan format tata letak elemen HTML pada halaman web.

Discover cara menggunakan CSS untuk merancang tata letak, warna, tipografi, animasi, dan responsivitas.
